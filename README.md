### Readme

#### Описание

Тестовое задание.

Сайт собранный с помощью Gulp. Скрипты написаны на JQuery.
Адаптивная вёрстка.Проверено Prettier.


#### Как запустить

1. Поочерёдный запуск задач для сборки проекта в build 
```bash
$ npm run build
```
2. Запуск Gulp для разработки в html-версии
```bash
$npm run html
```
